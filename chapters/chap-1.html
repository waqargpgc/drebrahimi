Chapter 1

THE EVOLUTION OF COMPUTER PROGRAMMING AND LANGUAGES: AN OVERVIEW OF C AND C++

It is important to familiarize yourself with the fundamentals of computer programming, computer languages, and their evolution. It may not be apparent to you at this time how this knowledge is going to help you to program in C/C++. It should, however, be noted that within the past fifty years, a multitude of programming concepts and methods has been introduced, and many different languages have been designed. Yet, they all still depend on a foundation that has remained relatively the same. In this chapter, you are going to encounter unfamiliar terminology. What I would like you to do is to skim through this chapter and grasp as much as you can, without attempting to understand everything fully. By examining the history of computer programming, it is hoped that you will be able to better understand the way programming and languages are shaped, and have an idea as to what to expect for the future of computer programming and languages. In order to give you a picture of computer programming and its abilities, I will introduce you to a very important program that is known as a search, written in both C and C++. You will then experience what I call the magic of programming, where the knowledge of building one program will lead you to build other programs without much effort. For example, you will see how a Telephone Operator Search Program can also act as a language translator, a dictionary, and can also easily become a bank or supermarket retrieval system. I will conclude the chapter by giving examples of pitfalls in C/C++ programming that may cause misconceptions and delay learning. These pitfalls, which are the result of hidden notions and unknown rules, are contributing factors to what I call the mystery of C/C++ programming. You should be aware that the problems these pitfalls cause have nothing to do with your intelligence, and that you will learn to solve them. By going through the foundations of programming and languages you will find it easier to build programs from scratch using C/C++ in the chapters that follow.

Why Program?

Historically, computers were developed to serve scientific communities. Today, computers have become integral to nearly every aspect of life. Understanding how programs operate helps you create applications that solve real-world problems.

Learning programming allows you to take control of the computer's "mind," making it an essential skill for solving modern challenges.

Are We Programmed?

Have you ever wondered if we, like computers, follow a program of events? Understanding programming is akin to understanding decision-making processes in our daily lives. It is the basis for how computers function and make choices based on user input.

Computer Programs as Pre-Agreed Rules

In ancient times, humans communicated using rudimentary symbols or signs. In the same manner, computers communicate based on pre-agreed rules such as binary coding (0s and 1s). Understanding these concepts is crucial for writing effective code.

Code Example: C Version of the Telephone Operator Program
#include <stdio.h>
        #include <string.h>
        
        int main() {
        char searchname[15], name[15], telephone[14];
        FILE *fp;
        printf("ENTER THE SEARCH NAME: ");
        scanf("%s", searchname);
        fp = fopen("data.in", "r");
        while (fscanf(fp, "%s %s", name, telephone) != EOF) {
            if (strcmp(searchname, name) == 0) {
                printf("%s\n", telephone);
                return 0;
            }
        }
        printf("NAME NOT FOUND\n");
        return 0;
        }

This program searches for a person's telephone number using their name. It's a simple yet effective way to demonstrate searching in C programming.

Code Example: C++ Version of the Telephone Operator Program
#include <iostream>
        #include <fstream>
        #include <string.h>
        using namespace std;
        
        int main() {
        char searchname[15], name[15], telephone[14];
        cout << "ENTER THE SEARCH NAME: ";
        cin >> searchname;
        ifstream fin("data.in");
        while (fin >> name >> telephone) {
            if (strcmp(searchname, name) == 0) {
                cout << telephone << endl;
                return 0;
            }
        }
        cout << "NAME NOT FOUND" << endl;
        return 0;
        }

The C++ version introduces a few enhancements, including better handling of input/output and more readable syntax, helping you see the advantages of C++ over C.
